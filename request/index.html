種目の追加申請

本コンテストサイトに新たな種目を追加したい方はこちらをご覧ください。

はじめに

まず、本コンテストの運営方針をご覧いただき、追加を希望する種目が運営方針に合致していることを確かめてください。

申請方法

申請に必要なもの

  • 申請する競技の名前(英語)
  • 申請する競技に使うパズルの写真(アイコン作成に使います)
  • 申請する競技に使うパズルのスクランブル方法またはスクランブル生成プログラム(プログラムの仕様については後述)

スクランブル生成プログラムの仕様

スクランブル生成プログラムを作ってくださる場合は以下の仕様にできる限りしたがって制作してください。

実行環境

  • Python 3.7.0
  • Raspbian 4.19
  • Raspberry Pi 3 B 1GB RAM

実行コマンドと出力形式

実行コマンドは
python3 scramble_[競技名].py
です。 標準出力に1つのスクランブルを改行なしで出力してください。
